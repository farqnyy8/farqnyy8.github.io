Game Rules

The game is designed to help people learn the words and recall ayaat of the Quran. For 2-4 players and for 2-10 rounds, you can select a section of the Quran to 'remember' (dh-k-r) and familiarize yourself with.

  • Say basmalah and have good intention
  • Assign a number of points for word (maybe 2) and verses (maybe 3)
  • When a word appears for a player, they should the meaning of the word. If they get say it correctly, ask them to recite a verse with that word for the section y'all selected. If they recite a verse, then give them all the points (2 + 3, from the example)
  • If they are only able to say the translation of the word, give them only the number of points y'all assigned to the word (2 points from the example).
  • If they are only able to recite a verse with the word, give them only the number of points y'all assigned for the verse (3 points from the example).
  • Enjoy playing! (and make dua for me)

Buttons

  • New Game: clears all previous data and displays menu for new game
  • Restart Game: clears scores and starts the game again with a different sequence of words
  • Game Rules: shows the game rules
  • Submit score: to put a player's score

Note: clicking on Restart after New Game would not bring the data from the former game back!
